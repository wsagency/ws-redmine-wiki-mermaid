# Changelog

All notable changes to ws-redmine-wiki-mermaid will be documented in this file.

## [1.0.0] - 2026-02-14

### Added
- Automatic Mermaid diagram rendering in wiki pages (fenced code blocks with `mermaid` language)
- Live split-pane editor modal with real-time preview
- Vendored Mermaid.js (no runtime CDN dependency)
- Support for all Mermaid diagram types: flowcharts, sequence, class, state, ER, gantt, pie, etc.
- "Insert Diagram" button in wiki edit toolbar
- Error display for invalid diagram syntax
